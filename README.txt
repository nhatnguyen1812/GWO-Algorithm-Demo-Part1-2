NOI DUNG MA NGUON:
File main.py bao gom ma nguon Python mo phong 4 thuat toan GWO va cac bien the:
1. Standard GWO
2. Binary GWO
3. Chaotic GWO
4. Hybrid GWO-PSO

INPUT & OUTPUT:
- Input: Cac tham so (So luong soi, Vong lap...) duoc cau hinh truc tiep trong bien CONF o dau file main.py.
- Output: Chuong trinh se in ket qua ra man hinh Console, DONG THOI tu dong xuat ra file "ket_qua_chay.txt" de tien theo doi va bao cao.

HUONG DAN CHAY:
1. Cai dat thu vien: pip install numpy
2. Chay lenh: python main.py
3. Kiem tra file "ket_qua_chay.txt" duoc sinh ra trong thu muc.
